# Test Technique – Djoli

## Prérequis
- Node.js ≥ 14
- Yarn ou npm

## Lancer le projet
```bash
#HTTPS
git clone https://github.com/Madjalia/react-front-technical-test.git
#SSH 
git clone git@github.com:Madjalia/react-front-technical-test.git
cd react-front-technical-test

npm install
npm run dev
# ou
yarn install
yarn dev
```
